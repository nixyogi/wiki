# Rust: Functions

Status: Courses
Tags: Rust

1. Functions are defined like this below 
    
    ```rust
    // Simple increment function
    fn function1(val: i32) -> i32 {
      newval = val + 1;
    }
    // Function with mutable arguments
    fn function1( mut val: i32) -> i32 {
      val = val + 1;
    }
    ```
